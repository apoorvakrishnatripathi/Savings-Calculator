1. Problem Statement:-

The program's goal is to assist users in estimating possible investment returns. It enables users to either:

Utilise a personalised investment calculator where they input their own sum, duration, and interest rate, or

Determine returns using a predetermined list of stocks and the interest rates that go along with them.

Both simple interest and compound interest computations are supported by the program. Additionally, it determines if the user is at least eighteen years old to make the investment.

2. Scope of the Project:-

The scope consists of:

Name, age, investment amount, years, and interest options are all entered by the user.

displaying the interest rates of a predetermined list of twenty stocks.

Calculating interest using:

Basic interest

Interest compound

Both, upon request

permitting two operating modes:

Mode of custom interest rate

Interest rate mode based on stocks

Verifying the user's age (must be older than 17).

3. Target Users:-

The application is intended for:

novice investors looking for a straightforward return estimation tool.

Financial computations are taught to students.

Anyone looking for a quick comparison between compound and simple interest?

users who want to investigate projected returns using a set list of stocks.

4. High-Level Features:-

* Module for User Input

gathers information about investments, interest preferences, age, and name.

* System for Stock Display

displays a list of twenty pre-selected stocks along with the interest rates for each. 

# Investing savings calculator.

* The Calculation Engine

Assistance:

Easy Calculation of Interest

Calculating Compound Interest

Dual-mode (both kinds of interest)

* Two Modes of Investment

Mode of Custom Investment

The user inputs their own interest rate.

Stock Mode Predefined

The user chooses a stock and applies the interest rate that goes with it.

* Validation of Age

Calculations are only possible if the age is greater than 17.

Conditional Reasoning

adjusts outputs according to the chosen mode and interest type.
